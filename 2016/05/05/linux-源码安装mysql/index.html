  • Linux MySQL

解压源码并安装

配置MySQL

先拷贝my.cnf到/etc/下面 在初始化

修改环境变量

1.vim ~/.bash_profile 在最后面添加
export PATH=$PATH:/usr/local/mysql/bin(你的安装bin目录)
:wq 保存退出
2.source ~/.bash_profile (环境变量生效)
